# Microeconomics

Papers on microeconomic theory, industrial organization, behavioral economics, and related fields.

## Format
Each paper entry follows this structure:
```
paper-name/
├── paper-name.pdf          # Full paper PDF
├── paper-name.bib          # BibTeX citation
└── metadata.json           # Title, authors, abstract, keywords, DOI
```
