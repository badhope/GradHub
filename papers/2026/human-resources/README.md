## Paper Format
Each paper entry follows this structure:
```
paper-name/
├── paper-name.bib          # BibTeX citation
├── metadata.json           # Title, authors, abstract, keywords, DOI, PDF URL
└── README.md               # (Optional) Paper notes
```

> **Note**: PDFs are linked via DOI/open-access URLs. Download directly from the provided links.
