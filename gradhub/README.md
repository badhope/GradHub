# GradHub - 毕业项目导航站

🎓 为计算机专业学生设计的毕业项目导航平台

## ✨ 特性

- 🗂️ 多级导航系统 - 专业分类、技术栈、项目类型
- 📱 响应式设计 - 完美支持桌面、平板、移动端
- 🎨 唯美亮色调 - 现代简洁的UI设计
- 🚀 快速定位 - 从选题到实现,一站式解决方案

## 🎯 快速开始

直接在浏览器中打开 `index.html` 即可查看。

## 📂 项目结构

```
gradhub/
├── index.html          # 主页面
├── css/                # 样式文件
│   ├── variables.css   # CSS变量
│   ├── base.css       # 基础样式
│   ├── components.css # 组件样式
│   ├── layout.css     # 布局样式
│   ├── animations.css # 动画效果
│   └── responsive.css # 响应式
└── js/
    └── main.js        # 主逻辑
```

## 🎨 技术栈

- HTML5
- CSS3 (CSS Variables, Flexbox, Grid)
- Vanilla JavaScript
- Google Fonts (Noto Sans SC, Inter)
